# 🩸 Intelligent Donor Matching and Rapid Communication System

A web-based system that enables blood donation camps to quickly identify eligible blood donors in emergency situations based on blood group, availability, and past donation history. The system also tracks blood donation camps and donor activity effectively. It even expresses gratitude to donors through personalized thank-you messages after they donate.

---

## 🚀 Features

- 🔍 **Intelligent Donor Matching**: Filters donors by blood group, eligibility (based on last donation date), and district.
- 📢 **Emergency Alert System**: Allows admins to send alerts to suitable donors during emergencies.
- 💌 **Gratitude Messaging**: Automatically sends personalized thank-you messages with the donor's name after a successful donation.
- 👤 **Donor Registration & Dashboard**: Donors can register, log in, and track their donation history and eligibility.
- 🗂 **Admin Panel**: View all registered donors, donation records, and organize blood donation camps.
- 🏥 **Camp Management**: Add, view, and manage details of past blood donation camps.
- 📜 **Eligibility Check**: System calculates if a donor is eligible based on the last donation date.

---

## 🛠️ Tech Stack

- **Frontend**: HTML, CSS, JavaScript
- **Backend**: PHP
- **Database**: MySQL (via XAMPP)
